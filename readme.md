# FAST API 

## Démarré l'env

```
.venv\Scripts\activate
```

## lancé le serveur de dev 
```
uvicorn app.main:app --reload
```
## Format auto 
```
pip install black
```
## mettre a jour le fichier de requirments avec les dépendances installé
```
pip freeze > requirements.txt
```
## mettre a jour pip
```
pip install --upgrade pip
```

## mettre les dependances
```
pip install -r requirements.txt
```
